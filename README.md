# BOE.ICU

## 概述

使用 Solid.js 和 UnoCSS 构建的现代简约个人主页。

## 预览

1. 中文界面
![中文](/preview_cn.png)
2. 英文界面
![英文](/preview_en.png)

## 特性

- 🎨 现代简约设计
- 🌓 明暗主题（支持跟随系统）
- 🌍 中英双语支持
- 🚀 基于 Solid.js 的极致性能
- 📱 完全响应式布局
- 🤖 AI 驱动开发
- 🎯 SEO 优化

## 技术栈

- 框架：Solid.js
- 样式：UnoCSS
- 路由：@solidjs/router
- 图标：通过 UnoCSS preset-icons 支持多个图标集
- 构建工具：Vite

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
 ```
